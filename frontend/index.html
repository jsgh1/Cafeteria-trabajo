Proyecto Full Stack: DB + Backend + Frontend

Una cafetería digital con Docker, Liquibase, PHP, Swagger y una interfaz visual.

Este sitio documenta los 3 repositorios del trabajo y permite usar el CRUD visual conectado al backend.

Abrir panel administrativo

Repo DB

PostgreSQL con 5 entidades, datos iniciales, rollbacks, Liquibase y GitHub Actions para bloquear merges con errores.

Repo Backend

API REST en PHP por capas: Models, Repositories, Services y Controllers. Incluye Swagger con 30 endpoints.

Repo Frontend

Landing page y CRUD visual en HTML, CSS, JavaScript y Axios conectado al backend.

Cómo levantar todo

  1. Descomprime cafeteria-full.zip.
  2. Ejecuta docker compose up --build.
  3. Espera a que Liquibase aplique tablas e inserts.
  4. Abre http://localhost:8080/docs para Swagger.
  5. Abre http://localhost:3000 para el frontend.

Entidades de la base de datos

Primarias: categorías y productos. Secundarias: clientes, pedidos y pedido_items.
